# Timed-quiz

## Description

The timed quiz is an application that evaluates basic HTML-CSS-JS knowledge, it gives you 10 seconds to answere each question and, for every wrong answere your time will be reduced, you can store your high scores with local storage.

## Usage

To use the Work Day Scheduler, follow these steps:

1. Open the quiz in your web browser.
2. Click the start button to begin the quiz.
3. Answer the questions presented to you.
4. If you answer a question incorrectly, time will be subtracted from the clock.
5. After answering all questions or when the timer reaches 0, the game is over.
6. You can save your initials and your score when the game is over.
